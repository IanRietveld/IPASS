void dht11::get_data ( )
inline

Read five bytes send by the sensor.

Function to read the 5 bytes sent by the sensor of which the first (integral RH data) and third (integral T data) byte contain data which are stored for later usage.
If the high pulse is ~28 microseconds then it's a 0 and if it's ~70 microseconds then it's a 1. (This can be visualized by using a logic analyzer.)
